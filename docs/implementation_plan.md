# SL Colonies Web Rebuild Plan

## Architecture
- Static-first website generated by Node build script into `dist/`.
- Content-first structure with editable JSON files in `src/content/`.
- Shared visual system and components rendered by `scripts/build.mjs`.
- Netlify deployment via `netlify.toml` using `npm run build`.

## Page Map
1. `/` Home
2. `/players/`
3. `/world-builders/`
4. `/systems/`
5. `/community/`
6. `/news/`
7. `/faq/`
8. `/get-started/`

## Design Direction
- Theme: grounded fantasy + crafted-world atmosphere.
- Color: forest green, ember copper, parchment, obsidian.
- Typography: dramatic serif for display + readable sans for body.
- Motion: subtle reveal and hover transitions only.

## Research Notes
- Intended: live Discourse API pull (`latest`, `categories`, topic endpoints).
